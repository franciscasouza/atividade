Elementos do fluxograma

Clique em um elemento para adicioná-lo na área de criação. Depois, selecione o bloco para editar texto, ligar ao próximo passo e organizar o fluxo.

Como usar

1. Adicione os blocos.
2. Clique em um bloco para editá-lo.
3. Em Próximo bloco, escolha a ligação.
4. Em decisão, informe também os caminhos Sim e Não.
5. Clique em Concluir para validar.
Dica: em Entrada, você pode escrever várias variáveis, por exemplo: nome, idade, nota1, nota2.

Área de criação

Editor visual simplificado para montar e validar fluxogramas.

Propriedades do bloco

Selecione um bloco para editar. O aluno pode alterar texto, definir conexões e observar o modelo de referência.

Exemplo orientado

Problema: verificar se o aluno foi aprovado.
Entrada: nome, nota
Decisão: nota >= 60?
Saída 1: Aluno aprovado
Saída 2: Aluno reprovado
Exemplo de várias variáveis: nome, idade, salario.
Exemplo de processo: media = (nota1 + nota2) / 2.
